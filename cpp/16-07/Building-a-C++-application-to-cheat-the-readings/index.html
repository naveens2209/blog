Define functions

calc():

This is a function which will solve this equation( ) with the variables provided. 

float calc(float t,float u,float l){
	return sqrt(t)/(2*l*sqrt(u));
}
ten():

This function will multiply the value of weight by the constant of acelleration due to gravity.

float ten(float x){
  return x*0.0098;
}
com():

This function returns the mean of the two length input.

float com(float x1, float x2){
  return(x1+x2)/200;
}
main():

Here this function will run through at the beginning and end of the program.

int main()
{
	int w;
	float n,t,u,l1,l2,l;
	cout<<"Calculate Frequency of AC from lengths on Sonometer!!\n";
	cout<<"What are the lengths?(cm)\nl.1:";
